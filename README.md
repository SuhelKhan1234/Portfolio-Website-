This project is a modern and responsive personal portfolio website built using HTML5 and CSS3 to showcase my web development skills, projects, and technical knowledge. It features an animated hero section with a typing effect, interactive project cards, a dedicated skills section displaying the technologies I have learned, a contact form, and a professional footer with social media icons. While developing this project, I applied modern CSS concepts such as Flexbox, CSS Variables, Positioning, Transitions, Transforms, Hover Effects, and Keyframe Animations to create an attractive and engaging user interface. Font Awesome was used for icons, and Typed.js was integrated to add dynamic text animation. This project helped me strengthen my understanding of responsive layouts, clean code organization, and front-end design principles. It reflects my learning journey in web development and demonstrates my ability to build visually appealing, responsive, and user-friendly websites using HTML and CSS.
